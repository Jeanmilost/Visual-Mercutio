========================================================================
    BIBLIOTHÈQUE MICROSOFT FOUNDATION CLASS : vue d'ensemble du projet zMediator
========================================================================


AppWizard a créé cette DLL zMediator pour vous. Cette DLL décrit
non seulement les bases de l'utilisation de Microsoft Foundation classes mais
est également un point de départ pour l'écriture de votre DLL.

Ce fichier contient un résumé du contenu de chacun des fichiers qui
constituent votre DLL zMediator.

zMediator.vcproj
    Il s'agit du fichier projet principal pour les projets VC++ générés à l'aide d'un Assistant Application. 
    Il contient les informations sur la version de Visual C++ qui a généré le fichier et 
    des informations sur les plates-formes, configurations et fonctionnalités du projet sélectionnées avec
    l'Assistant Application.

zMediator.h
    Il s'agit du fichier d'en-tête principal pour la DLL. Il déclare la classe
    CzMediatorApp.

zMediator.cpp
    Il s'agit du fichier source DLL principal. Il contient la classe CzMediatorApp.
zMediator.rc
    Il s'agit de la liste de toutes les ressources Microsoft Windows que le
    programme utilise. Elle comprend les icônes, les bitmaps et les curseurs qui sont stockés
    dans le sous-répertoire RES. Ce fichier peut être modifié directement dans Microsoft
    Visual C++.

res\zMediator.rc2
    Ce fichier contient les ressources qui ne sont pas modifiées par Microsoft 
    Visual C++. Vous devez placer toutes les ressources non modifiables par
    l'éditeur de ressources dans ce fichier.

zMediator.def
    Ce fichier contient des informations sur la DLL qui doivent être
    fournies pour qu'elle s'exécute avec Microsoft Windows. Il définit les paramètres
    tels que le nom et la description de la DLL. Il exporte également
    les fonctions de la DLL.

/////////////////////////////////////////////////////////////////////////////
Autres fichiers standard :

StdAfx.h, StdAfx.cpp
    Ces fichiers sont utilisés pour générer un fichier d'en-tête précompilé (PCH) 
    nommé zMediator.pch et un fichier de type précompilé nommé StdAfx.obj.

Resource.h
    Il s'agit du fichier d'en-tête standard, qui définit les nouveaux ID de ressources.
    Microsoft Visual C++ lit et met à jour ce fichier.
