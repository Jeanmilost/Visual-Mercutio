Team Editor

 

Généralités

Team Editor est l’outil central de la suite Monitor. Il permet de gérer les employés et les équipes, éditer les différents temps d’activité, de faire le lien entre les systèmes externes des entreprise et la base de données Monitor. Il permet également de lancer la génération des différents indicateurs en interrogeant la base de données Monitor

 

Choix de l’univers

Au lancement de l’application il faut indiquer quel univers est concerné par la session de travail. Les univers sont définis dans le fichier .config.

 

 

Sélectionnez l’univers désiré dans la liste déroulante.

 

Gestion des utilisateurs

Sélectionner le menu Edition-Utilisateurs.

 

 

Cet écran permet de gérer les utilisateurs au niveau de leurs données personnelles, et planification du temps de travail.

 

Onglet Données personnelles :

Sur cet onglet se trouvent les données personnelles de l’utilisateur.

Lors de la sauvegarde en cliquant sur sauver, si le champ mot de passe est vide, celui-ci ne sera pas modifié.

La case à cocher « capsules » concerne uniquement Webmessenger dans un développement futur pour autoriser ou non l’accès aux capsule de travail pour le travail nomade.

 

Onglet Config AIP

Cet onglet permet de gérer les données liées à l’ajustement personnel.

 

Il est important de saisir des mois complets étant donné que Monitor travail uniquement avec la notion de mois.

 

Onglet Config AILE

Cet onglet permet de gérer les données liées à l’ajustement personnel lié à l’équipe. C’est cette procédure qui permet d’affecter un utilisateur à une équipe à un taux donné.

 

Onglet Calendrier

Cet onglet permet de gérer le calendrier personnel de l’utilisateur. Il permet de saisir le nombre de jours de présence prévu par l’employé. Il est possible de gérer les vacances à ce niveau ou alors au niveau AILE.

 

Pour mettre à jour le calendrier il faut sélectionner les cases de la ligne « Présence prévue » et indiquer le nombre de jour prévu. La ligne Jours disponibles est à titre d’information. Il s’agit du calendrier standard de l’entreprise. Par défaut la présence prévue est égale aux jours disponibles.

 

Cliquer sur « Sauver planification » pour enregistrer le calendrier de l’utilisateur. Cet action est également obligatoire pour valider les configurations AIP et AILE.

 

Les boutons <<  et  >> permettent de changer l’année.

 

Gestion des équipes

Sélectionner le menu Edition-Equipes

 

 

 

Cet écran permet de gérer les équipes sur Monitor. Par défaut lors d’une publication d’un modèle de Conceptor vers Monitor, chaque groupe se voit attribuer une équipe portant le même nom et représentant le 100% de l’activité du groupe. La gestion des équipes de monitor permet de modifier ces équipes, leur pourcentage d’activité ainsi que de créer de nouvelles équipes sous un groupe.

 

Lors de l’ajout ou édition d’une équipe, il faut choisir le groupe parent dans la liste déroulante « Groupe Parent » et lui affecter un pourcentage. Le pourcentage total des équipes se trouvant sous un groupe ne peut pas dépasser 100%. Cette information est indiquée dans la zone de texte « Total du groupe ».

 

Onglet Calendrier Calculé

 

Cet onglet est uniquement présent à titre d’information. Il permet d’indiquer la capacité totale d’une équipe en récupérant automatiquement toutes les valeurs des utilisateurs membres de cette équipe. Le total tient compte de tous les critères AIP et AILE de chaque membre de l’équipe.

 

Onglet Calendrier Manuel

Le calendrier manuel permet d’affecter globalement pour chaque mois la capacité de l’équipe sans tenir compte des membres. Ceci permet de ne pas tenir compte de la configuration des membres mais uniquement de l’équipe dans sa globalité.

 

Le bouton « sauver la planification » permet de sauvegarder la configuration de l’année en cours.
