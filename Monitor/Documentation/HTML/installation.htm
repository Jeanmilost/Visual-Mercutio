Installation des logiciels Monitor

 

Environnement et Pré-requis

 

Monitor est composé d’un ensemble de logiciels utilisant différents environnements.

 

-          .Net Framework 2.0

-          Java 1.3 (pour les compatibilités des anciens développements et webservices Webmessenger.

-          Java 1.5

-          Apache Tomcat

-          Mysql 5.0

-          Microsoft Excel (génération des indicateurs)

 

Installation de l’environnement

 

-          Copier les 2 répertoires jdk se trouvant dans \Prérequis à la racine du disque dur C

o       C:\jdk1.3.1_02

o       C:\jdk1.5.0_06

-          Copier le répertoire Prérequis\Apache Group et son contenu sur c:\progam files\

-          Installer Mysql 5.0. Les fichiers d’installation se trouvent dans Prérequis\Mysql\mysql-5.0.22-win32.zip

-          Installer Mysql Administrator pour administrer Mysql. Les fichiers d’installation se trouvent dans Prérequis\Mysql\mysql-administrator-1.1.9-win.msi

-          Installer l’environnement .net Framework 2.0 en exécutant le fichier Prérequis\dotnetfx.exe

-          Installer le connecteur Mysql pour .net se trouvant dans Prérequis\mysql-connector-net-1.0.7.zip

-          Installer le connecteur ODBC pour mysql se trouvant dans Prérequis\mysql-connector-odbc-3.51.12-win32.msi

 

 

Installation des logiciels Monitor

 

-          Créer un nouveau répertoire de votre choix (Monitor

-          Copier le répertoire Logiciels\Team Editor dans le répertoire Monitor de votre choix

-          Copier le répertoire Logiciels\Soap Feeder dans le répertoire Monitor de votre choix

-          Copier le répertoire Logiciels\BatchMessages dans le répertoire Monitor de votre choix

-          Copier le répertoire Logiciels\XMLBuilder dans le répertoire Monitor de votre choix

 

Installation des bases des données

 

Trois bases de données sont proposées afin d’avoir trois univers d’utilisation différents.

Utiliser l’outil MYSQL Administrator pour remonter les bases de données se trouvant dans le répertoire Données\. Les fichiers .sql comprennent les scripts de création des bases

-          pilotage.sql

-          workflow.sql

-          simulation.sql
