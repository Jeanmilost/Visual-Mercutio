Team Editor – Mapping des systèmes externes

 

Généralités

Afin de pouvoir faire des requêtes permettant de comparer le prévisionnel et le réalisé, Monitor a besoin de connaître les activités réalisées par les processus de l’entreprise. Afin de récupérer les données, Monitor reçoit des « messages » depuis les systèmes des entreprises.

 

Afin de faire correspondre les utilisateurs, procédures et livrable des systèmes de l’entreprise à ceux de Monitor un outil de mapping à été ajouté à Team Editor.

 

Lors de la réception d’un message d’un système externe, Monitor va rechercher la correspondance dans le table de mapping pour connaître la procédure / livrable / utilisateur  / prestation concernée. Il est donc important que les tables de correspondances soit sans faute.

 

Menu Systèmes-Mappage

 

 

Sur cet écran, quatre onglets qui permettent de faire la correspondance entre les procédures, livrables, utilisateurs et prestations.

 

Dans les listes déroulantes du sommet, se trouvent les systèmes externes existants, ainsi que les Références des modèles déjà publiés sur la base de données Monitor.

 

Dans le tableau, les cellules blanches correspondent aux données de Monitor. Elles présentent le nom de la procédure ainsi que l’identifiant interne à Monitor. Pour faire une correspondance avec un système externe, il faut saisir dans les cellules vertes, l’identifiant de la procédure sur le système externe. La description est un champs facultatif

 

Onglet Livrables

Dans cet onglet le principe est le même. Cependant ne sont listé que les Livrables des procédures qui ont déjà été « mappées » dans l’onglet précédant.

 

 

Onglet Utilisateurs

Même procédé dans cet onglet, il faut faire correspondre le userID du système externe aux utilisateurs existants sur Monitor.

 

Il est possible de lier plusieurs Utilisateurs externe sous un même utilisateur Monitor. Pour faire cela il suffit de séparer les utilisateurs externe par une virgule (,).

 

 

Onglet Prestations

 

Au niveau des prestations Externe il s’agit d’un champs « numérique » correspondant au numéro de la prestation définie dans le système externe
